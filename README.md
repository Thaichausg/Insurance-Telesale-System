# Insurance Telesale System

Hệ thống quản lý telesale bảo hiểm chuyên nghiệp. Dự án được cấu trúc theo kiến trúc Module-based (Module-Driven Architecture) nhằm đảm bảo tính độc lập, dễ mở rộng và bảo trì.

## Cấu trúc dự án (Monorepo)

- `apps/web/`: Ứng dụng Frontend (Next.js)
- `apps/api/`: Ứng dụng Backend (NestJS)
- `docs/`: Tài liệu dự án
- `scaffolds/`: Các công cụ sinh code (code generators)

## Bắt đầu

Xem chi tiết trong thư mục `docs/`.
